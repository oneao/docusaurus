## Docker 安装

拉取镜像

```bash
docker pull redis:8.4-alpine
```

快速启动

```bash
docker run -d --name redis -p 6370:6379 redis:8.4-alpine --requirepass "xxx"
```

在容器启动后配置密码

```bash
# 拉取redis镜像
docker pull redis

# 启动容器
docker run -d -p 6366:6379 --name myredis2 redis

# 通过容器id，进入redis
docker exec -it myredis2 /bin/bash

# 运行redis客户端
redis-cli

# 查看redis的密码
config get requirepass

# 设置redis的密码
config set requirepass yourPassword

# 认证
auth yourPassword
```
