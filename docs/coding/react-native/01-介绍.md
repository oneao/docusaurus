[React Native](https://reactnative.dev/) 是一个开源框架，用于构建跨平台的移动应用。它允许开发者使用 React 来构建原生的 iOS 和 Android 应用。

主要特点：
- 跨平台开发：通过一次编写代码，React Native 可以在 iOS 和 Android 上运行，大大减少开发时间和成本。
- 原生性能：React Native 使用原生组件（如按钮、文本框等），提供接近原生应用的性能和用户体验。
- 热重载：开发者可以实时查看代码修改的效果，而无需重新加载整个应用。
- 组件化开发：通过 React 的组件化理念，开发者可以更高效地构建和维护应用界面。
- 丰富的生态系统：React Native 拥有大量的第三方库和工具，帮助开发者快速实现功能。
