## 设置页面背景色

在 `pages.json` 中设置 `backgroundColor` 并未生效。

设置全局背景色在 `App.vue` 中进行设置

```css
<!--此标签需要新建，不要和样式搞在一起-->
<style>
page {
  background-color: #f5f7fa;
}
</style>
```

设置单个页面背景色就在对应的页面中设置

```css
<!--此标签需要新建，不要和样式搞在一起-->
<style scoped>
page {
  background-color: red;
}
</style>
```
